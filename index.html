FAQ

AI가 쓴 글이 네이버 검색에 불이익을 받지 않나요?

마케케는 프로필별 문체 학습과 자연스러운 프롬프트 흐름을 바탕으로 사람이 다듬은 것처럼 읽히는 결과를 목표로 합니다. 블로그 생성에는 SEO 흐름도 함께 반영됩니다.

코딩이나 기술 지식이 없어도 사용할 수 있나요?

물론입니다! 기술을 모르는 분을 위해 설계되었습니다. API 키 발급도 앱 내 가이드를 따라하면 5분이면 됩니다.

여러 업체의 마케팅을 동시에 관리할 수 있나요?

네. 프로필 시스템으로 업체별 업종, 계정, 문체, 프롬프트, Chrome 프로필을 따로 저장할 수 있어 대행사 운영에 특히 잘 맞습니다.

영상 편집 프로그램 없이도 영상을 만들 수 있나요?

네. 마케케 안에서 나레이션 대본, TTS 음성, 자막, BGM, 슬라이드 영상, AI 영상 생성까지 같은 흐름으로 처리할 수 있습니다.

네이버 블로그 자동 발행이 진짜 되나요?

네, Chrome DevTools Protocol을 통해 SmartEditor에 직접 입력합니다. 로그인, 글쓰기, 이미지 삽입, 태그, 카테고리 설정, 발행까지 전부 자동입니다.

단일 작업도 가능한가요?

가능합니다. 단일 카드뉴스, 단일 스레드/X, 단일 4컷 스토리 작업공간을 따로 제공해 URL 수집, 참조 이미지 조정, 회차 생성, 분할 복사까지 바로 처리할 수 있습니다.

macOS와 Windows 모두 지원하나요?

네, Electron 기반이라 macOS와 Windows 모두 지원합니다. 동일한 기능을 사용할 수 있습니다.
