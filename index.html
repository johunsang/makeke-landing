FAQ

AI가 쓴 글이 네이버 검색에 불이익을 받지 않나요? +

자연스러움 옵션으로 쉼표율, 마침표율, 사람다움 수준을 조절하여 사람이 직접 쓴 것과 구별이 어려운 글을 생성합니다. SEO 최적화가 자동 적용되어 네이버 검색에 최적화됩니다.

코딩이나 기술 지식이 없어도 사용할 수 있나요? +

물론입니다! 기술을 모르는 분을 위해 설계되었습니다. API 키 발급도 앱 내 가이드를 따라하면 5분이면 됩니다.

여러 업체의 마케팅을 동시에 관리할 수 있나요? +

네, 프로필 시스템으로 각 업체별로 업종, 톤, 계정, 스케줄을 모두 따로 설정할 수 있습니다. 대행사에서 특히 유용합니다.

영상 편집 프로그램 없이도 영상을 만들 수 있나요? +

네, 마케케 안에서 모든 것이 해결됩니다. AI가 나레이션 대본, TTS 음성, 자막, BGM, 카메라 움직임까지 모두 자동 처리합니다.

네이버 블로그 자동 발행이 진짜 되나요? +

네, Chrome DevTools Protocol을 통해 SmartEditor에 직접 입력합니다. 로그인, 글쓰기, 이미지 삽입, 태그, 카테고리 설정, 발행까지 전부 자동입니다.

맥(Mac)과 윈도우(Windows) 모두 지원하나요? +

네, Electron 기반이라 macOS와 Windows 모두 지원합니다. 동일한 기능을 사용할 수 있습니다.
