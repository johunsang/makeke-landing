마케케 완벽 가이드

앱 설치부터 프로필 설정, 콘텐츠 생성, 플랫폼 업로드, 데이터 백업까지. 마케케를 처음 사용하는 분도 이 가이드만 따라하면 바로 시작할 수 있습니다.

설치 파일 다운로드 문의하기
📦

1. 설치 방법

macOS 설치

  1. 다운로드 -- makeke.kr에서 최신 .zip 파일을 다운로드합니다.
  2. 압축 해제 -- 다운로드된 ZIP을 더블클릭하면 Makeke.app이 나타납니다. 이 파일을 Applications(응용 프로그램) 폴더로 드래그합니다.
  3. 보안 경고 허용 -- 처음 실행 시 "확인되지 않은 개발자" 경고가 나타납니다. 시스템 설정 > 개인정보 보호 및 보안에서 "확인 없이 열기"를 클릭합니다.
  4. Gemini API 키 설정 -- 앱 실행 후 좌측 메뉴의 설정 페이지에서 Gemini API 키를 입력합니다.
macOS 설치 과정 스크린샷
macOS Gatekeeper 경고는 서명되지 않은 앱에서 정상적으로 발생합니다. 한 번 허용하면 이후에는 나타나지 않습니다.

Windows 설치

  1. 다운로드 -- makeke.kr에서 최신 .exe 설치 파일을 다운로드합니다.
  2. SmartScreen 경고 -- 실행하면 "Windows에서 PC를 보호합니다" 화면이 나타납니다. "추가 정보"를 클릭한 뒤 "실행"을 클릭합니다.
  3. 자동 설치 완료 -- 설치가 완료되면 마케케가 자동으로 실행됩니다.
Windows 설치 과정 스크린샷
Windows Defender SmartScreen 경고는 새로운 앱에서 정상적으로 나타납니다. "추가 정보"를 누르면 실행 버튼이 보입니다.

Chrome 확장 프로그램

마케케 앱을 실행하면 블로그 발행 및 브라우저 자동화에 필요한 Chrome 확장 프로그램이 자동으로 설치됩니다.

수동 설치가 필요한 경우

  1. Chrome 주소창에 chrome://extensions를 입력합니다.
  2. 우측 상단의 개발자 모드를 켭니다.
  3. "압축해제된 확장 프로그램을 로드합니다"를 클릭합니다.
  4. 마케케 설치 경로의 chrome-extension 폴더를 선택합니다.
확장 프로그램이 정상 설치되면 Chrome 도구 모음에 마케케 아이콘이 나타납니다.
🚀

2. 사용법

프로필 설정

마케케는 프로필 단위로 업체 정보, 스타일, 업로드 계정을 관리합니다. 먼저 프로필을 만들어야 콘텐츠를 생성할 수 있습니다.

  1. 프로필 추가 -- 좌측 메뉴에서 "프로필 관리"를 클릭하고 "프로필 추가"를 누릅니다.
  2. 기본 정보 입력 -- 업체명, 업종, 서비스/제품 설명, 대표 키워드를 입력합니다.
  3. Chrome 프로필 연결 -- 블로그/SNS 업로드에 사용할 구글 계정의 Chrome 프로필을 연결합니다.
  4. 콘텐츠 옵션 설정 -- 블로그 글 길이, 카드뉴스 스타일, 영상 옵션 등 생성 옵션을 설정합니다.
  5. 글쓰기 스타일 학습 -- 블로그/스레드 글을 수집하여 AI가 나만의 글쓰기 스타일을 학습합니다. (최대 100개 글 분석)
프로필 설정 화면 스크린샷
글쓰기 스타일 학습을 하면 AI가 여러분의 어투, 문체, 표현 방식을 반영하여 더 자연스러운 콘텐츠를 생성합니다.

콘텐츠 생성 (5단계)

마케케의 콘텐츠 생성은 5단계로 이루어집니다. 각 단계를 순서대로 진행하면 블로그, 카드뉴스, 영상, SNS 콘텐츠가 한 번에 만들어집니다.

1
키워드 검색 주제 입력 또는 트렌드에서 선택
2
소스 선택 뉴스/블로그/유튜브 소스 가져오기
3
프롬프트 편집 AI 프롬프트 확인 및 수정
4
콘텐츠 생성 블로그/카드뉴스/영상/SNS 동시 생성
5
업로드 네이버/스레드/트위터/틱톡/유튜브 발행

1단계: 키워드 검색

원하는 주제를 직접 입력하거나, 실시간 트렌드 목록에서 키워드를 선택합니다. 트렌드 분석 결과도 함께 확인할 수 있습니다.

키워드 검색 화면 스크린샷

2단계: 소스 선택

키워드와 관련된 뉴스, 블로그, 유튜브 영상이 자동 수집됩니다. 참고할 소스를 선택하면 본문 내용이 가져와지고, 이것이 AI 글 생성의 핵심 자료가 됩니다.

소스 선택 화면 스크린샷

3단계: 프롬프트 편집

AI가 수집된 소스를 바탕으로 생성 프롬프트를 자동으로 구성합니다. 필요에 따라 프롬프트를 직접 수정할 수 있습니다.

4단계: 콘텐츠 생성

블로그 글, 카드뉴스, 만화, 영상, SNS(스레드/트위터) 콘텐츠를 한 번에 생성합니다. 프로필 옵션에 따라 필요한 콘텐츠만 선택적으로 생성할 수도 있습니다.

콘텐츠 생성 화면 스크린샷

5단계: 업로드

완성된 콘텐츠를 검수한 뒤, 원하는 플랫폼에 자동으로 업로드합니다.

플랫폼별 업로드

마케케는 여러 플랫폼에 콘텐츠를 자동으로 업로드할 수 있습니다. 각 플랫폼마다 사전 설정이 필요합니다.

네이버 블로그 Chrome 프로필 연결 필요. 카테고리가 자동 설정되며 SmartEditor를 통해 발행됩니다.
스레드 (Threads) Chrome에서 threads.net에 미리 로그인이 필요합니다. 텍스트와 이미지 함께 게시됩니다.
트위터 (X) Chrome에서 x.com에 미리 로그인이 필요합니다. 트윗과 이미지가 자동 게시됩니다.
틱톡 (TikTok) Chrome에서 TikTok에 미리 로그인이 필요합니다. 영상 콘텐츠가 자동 업로드됩니다.
유튜브 (YouTube) 설정 페이지에서 YouTube OAuth 인증을 완료해야 합니다. 영상과 메타데이터가 함께 업로드됩니다.
인스타그램 (Instagram) 프로필 편집에서 인스타그램 ID/PW를 입력합니다. 카드뉴스 이미지가 자동 게시됩니다.
Chrome 기반 업로드(네이버, 스레드, 트위터, 틱톡)는 반드시 해당 사이트에 Chrome으로 로그인되어 있어야 합니다.

브라우저 자동화

AI에게 자연어로 브라우저 작업을 지시할 수 있습니다. 콘텐츠 생성 외에 다양한 리서치와 데이터 수집 작업에 활용할 수 있습니다.

사용 예시

  • "구글에서 OO 검색해서 가격 비교표 만들어줘"
  • "네이버 쇼핑에서 인기 상품 10개 정리해줘"
  • "경쟁사 블로그 최근 글 목록 가져와줘"
브라우저 자동화 화면 스크린샷
브라우저 자동화는 좌측 메뉴의 "자동화" 탭에서 사용할 수 있습니다.

3. 업데이트 방법

자동 업데이트

마케케는 앱 시작 시 자동으로 최신 버전을 확인합니다.

  1. 자동 감지 -- 앱 실행 시 서버에서 최신 버전 정보를 확인합니다.
  2. 다운로드 -- 새 버전이 있으면 백그라운드에서 자동 다운로드됩니다.
  3. 재시작 -- 다운로드가 완료되면 "지금 재시작" 버튼을 클릭하여 업데이트를 적용합니다.
업데이트 시 기존 데이터(프로필, 히스토리, 생성 결과물 등)는 모두 유지됩니다. 데이터는 앱 설치 경로가 아닌 외부 데이터 폴더에 저장되므로 앱이 교체되어도 안전합니다.
💾

4. 데이터 백업

데이터 저장 위치

마케케의 모든 데이터(프로필, 히스토리, 생성 결과물, 설정)는 아래 외부 데이터 폴더에 저장됩니다.

macOS

~/Library/Application Support/Makeke/

Windows

%APPDATA%/Makeke/

백업 방법

위 폴더를 통째로 복사하여 안전한 곳에 보관하면 모든 데이터가 보존됩니다.

  1. 마케케 앱을 완전히 종료합니다.
  2. 위 경로의 Makeke 폴더 전체를 ZIP으로 압축합니다.
  3. 외장 드라이브, 클라우드 등 안전한 곳에 보관합니다.

복원 방법

  1. 마케케 앱을 종료합니다.
  2. 백업해 둔 Makeke 폴더를 같은 위치에 덮어씁니다.
  3. 앱을 다시 실행하면 이전 데이터가 복원됩니다.
반드시 앱을 종료한 상태에서 백업/복원을 진행하세요. 실행 중에 폴더를 복사하면 데이터가 손상될 수 있습니다.

5. 자주 묻는 질문

Chrome 연결이 안 돼요
Chrome 확장 프로그램이 설치되어 있는지 확인합니다. chrome://extensions에서 마케케 확장이 활성화되어 있는지 확인하고, 비활성 상태라면 토글을 켜주세요. 그래도 안 되면 확장 프로그램을 제거 후 다시 설치해보세요.
스레드(Threads) 게시가 안 돼요
Chrome에서 threads.net에 로그인되어 있는지 확인합니다. 프로필에 연결된 Chrome 프로필이 실제로 스레드에 로그인한 프로필인지도 확인해주세요.
카드뉴스 디자인이 마음에 안 들어요
3단계 프롬프트 편집에서 원하는 스타일이나 색상, 레이아웃을 구체적으로 지시할 수 있습니다. 또한 프로필 편집에서 카드뉴스 스타일을 변경하면 전체적인 디자인 방향이 달라집니다.
API 비용이 궁금해요
마케케는 Gemini API를 사용합니다. 일반적인 사용량 기준으로 월 약 $5~$20 수준입니다. Google AI Studio에서 직접 API 키를 발급받아 사용하며, 사용량은 Google Cloud Console에서 확인할 수 있습니다.
네이버 블로그 발행이 실패해요
Chrome 프로필이 네이버에 로그인되어 있는지 확인합니다. 또한 네이버 블로그 SmartEditor 페이지가 정상적으로 열리는지 직접 Chrome에서 확인해보세요. 블로그 발행은 브라우저 자동화를 통해 이루어지므로 Chrome이 정상 동작해야 합니다.
글쓰기 스타일 학습은 어떻게 하나요?
프로필 편집 화면에서 "글쓰기 스타일 학습" 섹션을 찾아, 본인의 블로그나 스레드 URL을 입력합니다. 최대 100개의 글을 수집하여 어투, 문체, 표현 방식 등을 AI가 분석합니다.
앱이 실행되지 않아요
macOS에서는 시스템 설정 > 보안에서 앱 실행을 허용했는지 확인합니다. Windows에서는 방화벽에서 마케케의 내부 서버(포트 8800) 접근이 차단되지 않았는지 확인합니다. 문제가 지속되면 앱을 삭제 후 재설치해보세요(데이터 폴더는 유지됩니다).
여러 업체를 관리할 수 있나요?
네, 프로필을 여러 개 만들면 업체별로 스타일, 키워드, 업로드 계정을 별도로 관리할 수 있습니다. 좌측 상단에서 프로필을 전환하면 됩니다.

더 궁금한 점이 있으신가요?

가이드에서 해결되지 않는 문제는 고객지원 페이지를 통해 문의해주세요.
