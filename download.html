설치 방법

macOS

  1. 다운로드한 .zip 파일을 더블클릭하여 압축 해제
  2. Makeke.app을 응용 프로그램 폴더로 이동
  3. 처음 실행 시 "확인되지 않은 개발자" 알림이 뜨면
    시스템 설정 → 개인정보 보호 및 보안 → "확인 없이 열기" 클릭
  4. 앱이 자동으로 Python 서버를 시작하고 Chrome에 확장 프로그램을 설치합니다

Windows

  1. 다운로드한 .exe 파일을 실행
  2. Windows Defender SmartScreen 경고가 뜨면
    "추가 정보" → "실행" 클릭
  3. 설치 마법사의 안내를 따라 설치 완료
  4. 앱이 자동으로 Python 서버를 시작하고 Chrome에 확장 프로그램을 설치합니다

자동 업데이트

새 버전이 나오면 앱 실행 시 자동으로 감지하고 다운로드합니다.
"지금 재시작" 버튼을 누르면 바로 업데이트되고, "나중에"를 누르면 다음 종료 시 자동 적용됩니다.
수동 업데이트가 필요하면 이 페이지에서 최신 버전을 다시 다운로드하세요.
